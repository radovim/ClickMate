This project is an ESP8266-based device that hosts an HTTPS server, allowing users to schedule actions remotely via a web interface. The device controls a servo motor to physically press a button at specified times, making it ideal for automating tasks such as turning on appliances, triggering keypads, or pressing switches.
Features:
- Secure HTTPS server for remote scheduling
- Web-based timer configuration
- Servo motor for physical button pressing
- Compact and WiFi-enabled
